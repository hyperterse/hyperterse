## System Requirements

| Requirement | Minimum |
|---|---|
| Operating system | Linux (amd64, arm64), macOS (amd64, arm64), Windows (amd64) |
| Bun or Node.js | Bun 1.0+ or Node.js 18+ (only required if route scripts import npm packages) |

The Hyperterse binary is a statically linked Go executable with no runtime dependencies. TypeScript bundling uses the embedded esbuild compiler â€” no external JavaScript runtime is needed to serve built manifests.

---

## Install via Script

```bash
curl -fsSL https://hyperterse.com/install | bash
```

## Install via Homebrew

```bash
brew install hyperterse/tap/hyperterse
```

## Install via npm

```bash
npm install -g hyperterse
```

## Install via Bun

```bash
bun install -g hyperterse
```

---

## Verify

```bash
hyperterse --version
```

## Upgrade

```bash
hyperterse upgrade
```
