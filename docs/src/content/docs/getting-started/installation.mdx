Hyperterse is a CLI tool for creating database APIs. Choose your preferred installation method below.

## Quick install

<Tabs>
  <TabItem label="cURL">
```bash
curl -fsSL https://hyperterse.com/install | bash
```

  </TabItem>
  <TabItem label="NPM">
```bash
npm install -g hyperterse
```
  </TabItem>
  <TabItem label="Bun">
```bash
bun add -g hyperterse
```
  </TabItem>
  <TabItem label="Homebrew">
```bash
brew install hyperterse/tap/hyperterse
```
  </TabItem>
</Tabs>

## Verify installation

After installation, verify that Hyperterse is installed correctly by running the following command. It should output the version number of the installed Hyperterse binary:

```bash
hyperterse --version
```
